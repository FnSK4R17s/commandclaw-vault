# BOOTSTRAP.md - First Run

Welcome to your vault. This is your first session.

## Steps

1. Read every file in this directory to understand your workspace structure.
2. Fill in `IDENTITY.md` — pick a name, creature type, vibe, and emoji.
3. Fill in `USER.md` with whatever your human has told you about themselves.
4. Install skills: `npx skills add <skill-name>` (from the [commandclaw-skills](https://github.com/FnSK4R17s/commandclaw-skills) repo).
5. Read all installed skills in `.agents/skills/` to know what you can do.
6. Create today's daily note at `memory/YYYY-MM-DD.md`.
7. Delete this file — you won't need it again.

## Notes

- Your vault is a Git repository. Changes are versioned automatically.
- You can open this vault in Obsidian — plugins are pre-configured.
- Skills live in `.agents/skills/` — install them with `npx skills add`, not manually.
- Memory files are markdown. Write freely — Git tracks everything.
